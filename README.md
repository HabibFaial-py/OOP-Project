# 🏥 Hospital Management System (OOP Project)

## 📖 Project Description
The Hospital Management System is an Object-Oriented Programming (OOP) based application designed to manage hospital operations efficiently. It helps organize patient records, doctor details, appointments, and billing processes.

## 🔍 System Use Cases
1. **Patient Registration**: Add new patients with personal and medical details.
2. **Doctor Management**: Manage doctor information such as specialization and availability.
3. **Appointment Scheduling**: Book, update, or cancel appointments.
4. **Medical Records Management**: Store patient history, diagnoses, and treatments.
5. **Billing System**: Generate and manage billing information.
6. **Search and Display Records**: Retrieve patient or doctor data efficiently.
